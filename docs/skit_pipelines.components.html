skit_pipelines.components.train_voicebot_xlmr module#

train_voicebot_xlmr(data_path, model_path, label_column='intent_y', utterance_column='utterances', model_type='xlmroberta', model_name='xlm-roberta-base', num_train_epochs=10, early_stopping_patience=3, use_early_stopping=False, early_stopping_delta=0, max_seq_length=128, learning_rate=4e-05)[source]#

Warning

This component runs in a python 3.8 environment.

Parameters

  • data_path (InputPath) Path to load training dataset.

  • model_path (OutputPath) Path to save trained model.

  • label_column (str, optional) Column where intents are present, defaults to intent_y

  • utterance_column (str, optional) Column where ASR utterances are present, defaults to utterances

  • model_type (str, optional) BERT model type, defaults to xlmroberta

  • model_name (str, optional) BERT model subtype, defaults to xlm-roberta-base

  • num_train_epochs (int, optional) Number of epochs to train on, defaults to 10

  • early_stopping_patience (int, optional) Consecutive number of times early stopping threshold should be met, defaults to 3

  • use_early_stopping (bool, optional) Use early stopping to stop training if loss delta is stagnant, defaults to False

  • early_stopping_delta (float, optional) The difference between losses, defaults to 0

  • max_seq_length (int, optional) Truncate input after these many characters, defaults to 128

  • learning_rate (float, optional) Weight update parameter, defaults to 4e-5
